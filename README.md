# Colecciones y Sistema de Stock
* **Paquete:** `trabajo.practicon6`
* **Descripción:** Implementación de un sistema de inventario utilizando la colección `List<Producto>`. Incluye 10 métodos de alto volumen para buscar, filtrar por precio/categoría y gestionar el stock. La clase `CategoriaProducto` utiliza enumeraciones para tipado seguro.
